This git repository includes the TEI documents of the full text of 
82 volumes of the Archives Parlementaires created for the 
 French Revolution Digital Archive
by Stanford University Libraries.  These volumes are in the public
domain.

The directory structure and directory and file names reflect the 
Stanford University Libraries Digital Object Repository identifiers,
called druids (Digital Repository Universal Identifier).  Included
are files mapping Archive volume numbers to druids and vice versa.
